# Build from sources

Follow the steps below to build MediaSort from sources.

<Callout emoji="❌">
  **MediaSort** is not yet officialy supported on **Linux** or **macOS**. Try it at your own risk.
</Callout>

## Prerequisites

- [Git](https://git-scm.com/) (latest version)
- [Rust](https://www.rust-lang.org/tools/install) (1.56.1 or later)
- [Cargo](https://doc.rust-lang.org/cargo/getting-started/installation.html) (generaly installed with Rust)

## Builds steps

<Steps>
### Clone the repository
Open your terminal and run the following command:

```bash copy
git clone https://github.com/Angel-2180/MediaSort.git
cd MediaSort
```

### Build the project
Run the following command to build the project:

```bash copy
cargo build --release
```

### Run the project
After the build is complete, you can run the project using the following command:

```bash copy
MediaSort --help
```
</Steps>

<Callout emoji="🚀">
  It's recommanded to add the executable to the PATH to run it from anywhere.
  * [How to add to PATH on Windows 10](https://www.architectryan.com/2018/03/17/add-to-the-path-on-windows-10/)
  * [How to add to PATH on linux/macOS](https://stackoverflow.com/questions/14637979/how-to-permanently-set-path-on-linux-unix)
</Callout>

## Updating the project

To update the project, you can run the following commands:

<Steps>
### Open the project directory
```bash copy
cd MediaSort
```

### Pull the latest changes

<Callout emoji="🔨">
  Make sure you have no local changes before pulling the latest changes.
</Callout>

Use git to pull the latest changes from the upstream repository:

```bash copy
git pull
```

### Rebuild the project

Using cargo, you can rebuild the project with the following command:

```bash copy
cargo build --release
```

### Run the project

After the build is complete, you can run the `help` command to see if everything is working correctly:

```bash copy
MediaSort --help
```
</Steps>

## Troubleshooting

If you encounter any issues while building the project, please address them in the [issues section](https://github.com/Angel-2180/MediaSort/issues) on GitHub. We will do our best to help you resolve them.
